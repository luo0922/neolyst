# ui-prototype-coming-soon 规格

## Purpose

TBD（原型阶段：为未实现功能提供统一落点 `/coming-soon`）。

## Requirements

### Requirement: 系统提供 Coming Soon 占位页
系统 SHALL 提供统一占位页用于承载未实现功能的导航落点。

#### Scenario: 访问占位页
- **WHEN** 用户访问 `/coming-soon`
- **THEN** 页面 MUST 展示标题 `Coming Soon`
- **AND** 页面 MUST 展示提示 `This feature is coming soon`
